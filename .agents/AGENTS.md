# Custom Behavior Rules for SpeakUp Project

- **Documentation First:** Trước khi code một trang hay tính năng mới (ví dụ như trang Dashboard), LUÔN LUÔN tự động tạo một file mô tả dựa trên template (nằm trong `docs/templates/feature-spec-template.md`) và lưu vào thư mục `docs/03-features/`. Không bao giờ bỏ qua bước phân tích và lập tài liệu này.

- **State Management:** BẮT BUỘC ĐỌC file `docs/PROJECT_STATE.md` ngay ở bước đầu tiên khi bắt đầu làm việc để hiểu hiện trạng dự án. SAU KHI hoàn thành một tính năng lớn, BẮT BUỘC phải cập nhật file này để lưu trữ tiến độ cho các phiên làm việc sau.
